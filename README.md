# TrackerTracker

Multi-project scrum UI for Pivotal Tracker.

## Installation

1. Install latest [Node binary for OSX](http://nodejs.org/), then run...

```sh
brew install redis
git clone git@github.com:intentmedia/TrackerTracker.git
cd TrackerTracker/app
npm install
```

## Running the server

```sh
cd TrackerTracker/app
node app
```

## Run redis

```sh
redis-server
```
